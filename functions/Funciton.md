function : Function is  the set of code or block of code that executes the spacific tast.

Types of function java script:
                              |- Traditional function
                              |- arrow function
                                 |- function with return value
                                 |- function with parameter



Traditional function - 
                    syntax: function function_name(){
                        ### code or statement
                    }

                    function_name(); ### function calling ###

//<==========================================================================================>

Arrow function -  don't need to use function keyword

                    syntax: function_name = () =>{
                        ### code or statement
                    }

                    function_name(); ### function calling ###
